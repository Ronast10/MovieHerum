# 🎬 MovieHerum

A full-stack movie discovery and review platform built with the MERN stack.

## Features
- 🔐 JWT Authentication with email verification
- 🔑 Google OAuth login
- 🎬 Movie browsing powered by TMDB API
- ⭐ Rate and review movies
- 📌 Personal watchlist
- 👤 User profiles with bio editing
- 📧 Email verification via Nodemailer

## Tech Stack
**Frontend:** React 18, Vite, Tailwind CSS, Zustand, React Router  
**Backend:** Node.js, Express, MongoDB, Mongoose  
**Auth:** JWT, Passport.js, Google OAuth 2.0  
**Data:** TMDB API  

## Setup

### Backend
```bash
cd server
npm install
npm run dev
```

### Frontend
```bash
cd client
npm install
npm run dev
```
